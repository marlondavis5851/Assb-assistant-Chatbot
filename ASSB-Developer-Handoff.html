3. Deploy to GitHub Pages (Free, ~10 minutes)

  1. Go to github.com → click New Repository
    Name: assb-assistant · Visibility: Public
  2. Click "uploading an existing file" → upload all four HTML files
    ⚠️ Rename anguilla-ssb-chatbot.html to index.html
  3. Go to Settings → Pages → Source: Deploy from branch → main → / (root) → Save
  4. Wait ~2 minutes → your bot is live at:
    https://[your-github-username].github.io/assb-assistant/
Recommended for ASSB: Host at assistant.ssbai.com as a subdomain of the official site. This gives public trust and a clean URL. Ask your existing hosting provider to point that subdomain to the GitHub Pages URL.

5. How to Add a New Knowledge Base Topic

Find const KB = [ in the file. Add a new entry:

// ADD INSIDE KB = [ ... ]
{
  topic: "Your Topic Name",
  keywords: [
    "word people type",
    "phrase they might use",
    "another variation",
    "common question fragment"
  ],
  answer: `<strong>Topic Title</strong><br>
Opening sentence answering the question directly.<br><br>
<ul>
  <li>Key point one</li>
  <li>Key point two</li>
  <li>Visit <a href="https://www.ssbai.com" target="_blank">ssbai.com</a> for more</li>
</ul>`
},

Then add a matching TOPIC_VOICE entry:

// ADD INSIDE TOPIC_VOICE = { ... }
"Your Topic Name": {
  open: "Great question! Here's what you need to know —",
  close: "Feel free to ask anything else. 😊"
},
Keyword tips: Use lowercase phrases people actually type — not formal terms. E.g. "lost my job" not "cessation of employment". More keywords = better matching.

6. How to Change Login Credentials

// Find this block in the JavaScript section:
const USERS = {
  "assb":    "Ssbai2026!",   // username: "password"
  "admin":   "Assb@2026!",
  "marlon":  "Davis2026!"
};

Add, remove, or update entries. Usernames are automatically lowercased. Passwords are case-sensitive.

⚠️ Security note: This is client-side auth — credentials are visible in the page source. This is acceptable for a staff-facing internal tool. For a public-facing deployment, move authentication to a server-side backend (Node.js, PHP, or Cloudflare Workers with KV storage).

8. Phase 2 — AI-Powered Free-Form Answers

Currently the bot answers from a curated knowledge base (Phase 1). To add Claude AI for free-form questions not covered by the KB, deploy a Cloudflare Worker as a secure relay:

Step 1 — Create a Cloudflare Worker

// worker.js — deploy at dash.cloudflare.com → Workers
export default {
  async fetch(request, env) {
    if (request.method === 'OPTIONS') {
      return new Response(null, {
        headers: {
          'Access-Control-Allow-Origin': '*',
          'Access-Control-Allow-Methods': 'POST',
          'Access-Control-Allow-Headers': 'Content-Type'
        }
      });
    }
    const body = await request.json();
    const response = await fetch('https://api.anthropic.com/v1/messages', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': env.ANTHROPIC_API_KEY,
        'anthropic-version': '2023-06-01'
      },
      body: JSON.stringify(body)
    });
    const data = await response.json();
    return new Response(JSON.stringify(data), {
      headers: {
        'Content-Type': 'application/json',
        'Access-Control-Allow-Origin': '*'
      }
    });
  }
};

Step 2 — Add API key as a secret

wrangler secret put ANTHROPIC_API_KEY
# Paste your Anthropic API key when prompted

Step 3 — Update the chatbot

In the handleSend() function, after the KB match fails, add an API fallback:

// After: if(matches && matches.length) { ... } else {
// REPLACE the else block with:
} else {
  // Try AI for questions outside the knowledge base
  try {
    const res = await fetch('https://your-worker.workers.dev', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        model: 'claude-sonnet-4-6',
        max_tokens: 600,
        system: 'You are the ASSB Virtual Assistant. Answer only questions about Anguilla Social Security. Be warm and conversational.',
        messages: [{ role: 'user', content: txt }]
      })
    });
    const data = await res.json();
    addBot(data.content[0].text);
  } catch(e) {
    addBot('I don\'t have that detail — please call 264 497 2201/2.');
  }
}
Cost estimate: Cloudflare Workers free tier = 100,000 requests/month. Anthropic API = ~$0.003 per question. For 1,000 questions/month ≈ $3/month.
